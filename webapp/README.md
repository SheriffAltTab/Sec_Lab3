# Sec Lab 3 - Web Trialware Application

Цей веб-застосунок реалізує вимоги лабораторної роботи №3:

- модель розповсюдження: `Trialware`;
- обмеження: блокування після 30 днів від першого запуску (імітація дати встановлення);
- шифрування ключа розблокування: `шифр Віженера`.

## Як запустити

1. Відкрити `index.html` у браузері.
2. Використовувати функції редактора в Trial-режимі.
3. Якщо Trial завершився - активувати ліцензію ключем.

## Ліцензійний ключ

Статичного одного ключа немає. Застосунок приймає складні ключі, які
генеруються окремою програмою `../KEYGEN`.

Формат: 7 блоків по 5 символів (алфавітно-цифровий ключ), наприклад:
`ABCDE-FGHIJ-KLMNO-PQRST-UVWXY-12345-67890`

Перевірка включає:
- шифрування/дешифрування Віженера;
- перевірку службового payload;
- перевірку підпису ключа (вбудований signing salt).

## Реалізовані функції (мінімум 10)

1. Новий документ.
2. Відкриття файлу.
3. Збереження файлу.
4. Друк.
5. Копіювання в буфер обміну.
6. Очищення тексту.
7. Пошук підрядка (кількість входжень).
8. Заміна всіх входжень.
9. Перетворення у верхній регістр.
10. Перетворення у нижній регістр.
11. Реверс тексту.
12. Видалення зайвих пробілів.
13. Статистика (символи, слова, рядки).
14. Сортування рядків.
15. Видалення порожніх рядків.
