# KEYGEN - Sec Lab 3

Окрема програма генерації ліцензійних ключів для `webapp`.

## Запуск

Відкрийте `index.html` у браузері.

## Принцип

- формується payload довжиною 25 символів;
- payload шифрується методом Віженера;
- обчислюється підпис на 10 символів;
- ключ формується у 7 блоків по 5 символів.

Ключі з цього генератора валідні для активації у `../webapp`.
